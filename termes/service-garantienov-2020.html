  1. Éligibilité :  Pour bénéficier de l'Offre, le client doit :
  • être âgé d'au moins 18 ans ; et
  • être titulaire d'un compte Deliveroo.

Deliveroo se réserve le droit d'exiger à tout moment une preuve d'éligibilité à l'Offre.

  1. Offre : Lorsqu'une Commande éligible est livrée avec plus de trente (30) minutes de retard à compter de l'heure de livraison maximale estimée indiquée au client dans la confirmation de l'acceptation de sa commande ("Commande Retardée"), Deliveroo offre un code promotionnel à valoir sur une prochaine commande d'une valeur de :
  • quinze (15) euros pour tout nouveau client (c'est à dire pour tout client inscrit sur la plateforme dans les 56 jours calendaires précédant la Commande Retardée);
  • dix (10) euros pour tous les autres clients (c'est à dire pour tout client inscrit sur la plateforme depuis plus de 56 jours calendaires précédant la Commande Retardée);

(ci-après, "Code Promotionnel")

  1. Commandes éligibles : Une commande est éligible à l'attribution du Code Promotionnel dans le cadre de l'Offre si les critères cumulatifs suivants sont remplis :
  • Commande livrée dans le département 75,dans une sélection de villes d'île-de-France (dont le code postal est précisé en bas de page(1)), à Valenciennes ou à La-Roche-sur-Yon; et
  • Commande avec livraison immédiate (les commandes programmées ne sont pas éligibles à l'Offre) ; et
  • Commande livrée par les livreurs partenaires de Deliveroo (les commandes dont la livraison est effectuée par le restaurant ne sont pas éligibles) ; et
  • Commande dont le panier (hors frais de livraison, hors frais de services et hors pourboire) est inférieur ou égal à 100€ TTC
  1. Conditions d'utilisation du Code Promotionnel :
  • Le Code Promotionnel sera envoyé par Deliveroo au client dans un délai de 2 jours ouvrables à compter de la livraison de la Commande.
  • Le Code Promotionnel devra être entré dans la case prévue à cet effet par le client sur son compte Deliveroo. Il devra être utilisé sur une commande effectuée dans un délai de 28 jours ouvrables suivant son envoi. Passé ce délai, le Code Promotionnel expirera et ne sera plus valable.  
  • Le Code Promotionnel est valable sur toute commande sans valeur minimum, et pourra être utilisé pour payer le contenu du panier et/ou les frais de service et/ou les frais de livraison. Si la valeur de la commande est inférieure au montant du Code Promotionnel, le solde du Code Promotionnel sera perdu.
  • Le Code Promotionnel est non-cumulable, non-transférable, non-remboursable, et non-échangeable.

Annulation et modification : Deliveroo se réserve le droit de modifier, suspendre ou annuler l'Offre et/ou les présentes Conditions s'il le juge nécessaire ou approprié, y compris en cas de violation réelle ou anticipée de la loi applicable ou si la modification, la suspension ou l'annulation est nécessaire en raison d'un événement échappant au contrôle raisonnable de Deliveroo. Dans ce cas, Deliveroo en informera le Client par tout moyen. Toute modification, suspension ou annulation n'aura pas d'impact sur les Codes Promotionnels déjà émis et encore valables.
